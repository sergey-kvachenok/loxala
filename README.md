# Loxala

## 📦 Setup

```bash
# Clone repo
git clone <repository-url>
cd loxala

# Install dependencies
npm install
```

## 🛠️ Development

```bash
# Start server
npm run dev

# Check types
npm run type-check

# Run linter
npm run lint

# Fix linter errors
npm run lint:fix
```

## 🏗️ Build

```bash
# Создать production сборку
npm run build

# Production preview
npm run preview
```
